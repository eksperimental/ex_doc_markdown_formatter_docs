# ArithmeticError exception
(Elixir v1.18.0-dev)

An exception raised on invalid arithmetic operations.

For example, this exception is raised if you divide by `0`:

    iex> 1 / 0
    ** (ArithmeticError) bad argument in arithmetic expression: 1 / 0
